Направления в IT

Исследуй основные области разработки и найди свою

Web-разработка

Frontend Backend

Создание веб-сайтов и приложений. Frontend — то, что видит пользователь. Backend — логика на сервере.

Задачи

  • Верстка макетов (HTML/CSS)
  • Логика интерфейса (JS)
  • Работа с базами данных
HTML/CSS React Node.js
Подробнее 80k - 300k ₽

Mobile Разработка

iOS/Android

Создание приложений для iOS и Android. Можно разрабатывать нативные приложения (Swift/Kotlin) или кроссплатформенные (Flutter/React Native). Мобильная разработка — это не только приложения, но и интеграция с аппаратными возможностями телефонов: камерой, GPS, сенсорами.

Swift Kotlin Flutter
Подробнее 100k - 350k ₽

Data Science

AI/ML

Наука о данных — это извлечение знаний и инсайтов из структурированных и неструктурированных данных. Data Scientist не только строит модели, но и очищает данные, проводит статистический анализ и визуализирует результаты. Это мост между математикой, программированием и бизнес-задачами.

Python PyTorch SQL
Подробнее 120k - 450k ₽

GameDev

3D графика мобильные игры

Создание компьютерных и мобильных игр — от инди-проектов до AAA-хитов. Включает программирование игровой логики, работу с графикой (2D/3D), физикой, звуком и AI для NPC. Это одна из самых творческих областей в IT.

C++ C# Unity Unreal Engine Lua Python OpenGL/DirectX Blender/Maya
Подробнее 120k - 450k ₽
