Направления в IT

Выбери свой путь: от создания сайтов и мобильных приложений до реверс-инжиниринга и дизайна интерфейсов.

Web-разработка

Создание веб-сайтов и приложений. Frontend — визуальная часть. Backend — серверная логика.

Что делают?

  • Верстка интерфейсов (HTML/CSS)
  • Разработка логики (JavaScript)
  • Работа с базами данных и API

Стек технологий

HTML/CSS JavaScript React Node.js

Плюсы

  • Легко начать
  • Огромное количество вакансий

Минусы

  • Высокая конкуренция на старте
  • Технологии быстро меняются
Подробнее 80k - 300k ₽

Mobile Разработка

Создание приложений для iOS и Android с использованием нативных или кроссплатформенных технологий.

Что делают?

  • Проектирование UI для смартфонов
  • Работа с камерой, GPS и сенсорами
  • Публикация в App Store / Google Play

Стек технологий

Swift Kotlin Flutter React Native

Плюсы

  • Продукт всегда в кармане
  • Понятный результат работы

Минусы

  • Сложности с публикацией в сторы
  • Зоопарк устройств (особенно Android)
Подробнее 100k - 350k ₽

GameDev

Создание игр для ПК, консолей и мобильных платформ. Работа с графикой, физикой и игровой логикой.

Что делают?

  • Написание игровых механик
  • Работа с 3D/2D движками
  • Создание ИИ для врагов

Стек технологий

C++ C# Unity Unreal Engine

Плюсы

  • Работа мечты для геймеров
  • Творческая свобода (в инди)

Минусы

  • Частые переработки (кранчи)
  • Сложная математика
Подробнее 120k - 450k ₽

Data Science

Извлечение знаний из массивов информации. Статистический анализ, машинное обучение и нейросети.

Что делают?

  • Сбор и очистка больших данных
  • Обучение ИИ и нейросетей
  • Построение предиктивных моделей

Стек технологий

Python PyTorch TensorFlow SQL

Плюсы

  • Работа на острие прогресса
  • Очень высокие зарплаты

Минусы

  • Нужна сильная математика
  • Много рутины с очисткой данных
Подробнее 120k - 450k ₽

Reverse Engineering

Исследование готового ПО для понимания принципов его работы, поиска уязвимостей и анализа памяти.

Что делают?

  • Анализ бинарных файлов и декомпиляция
  • Исследование работы античитов и защит
  • Поиск уязвимостей (Exploit Development)

Стек технологий

C/C++ Assembly IDA Pro x64dbg Cheat Engine

Плюсы

  • Высочайший уровень знаний ПК
  • Уникальная и элитная ниша

Минусы

  • Порог входа — космос
  • Нужно знать архитектуру CPU
Подробнее 200k - 600k+ ₽

Cybersecurity (ИБ)

Защита систем, сетей и программ от цифровых атак. Этичный хакинг и аудит безопасности.

Что делают?

  • Пентестинг (поиск дыр в системах)
  • Настройка защиты серверов и сетей
  • Расследование киберинцидентов

Стек технологий

Python Bash Kali Linux Metasploit Wireshark

Плюсы

  • Всегда актуально
  • Чувствуешь себя хакером

Минусы

  • Огромная ответственность
  • Нужно знать сети от и до
Подробнее 150k - 500k ₽

DevOps Engineering

Автоматизация процессов разработки, настройки серверов и развертывания приложений (CI/CD).

Что делают?

  • Настройка пайплайнов для деплоя
  • Управление облачной инфраструктурой
  • Настройка серверов и контейнеров

Стек технологий

Linux Docker Kubernetes GitLab CI Terraform

Плюсы

  • Очень высокие зарплаты
  • Минимум рутинной работы

Минусы

  • Огромный зоопарк технологий
  • Часто нужно быть на связи 24/7
Подробнее 250k - 600k ₽

UI/UX Design

Проектирование логики пользовательского опыта (UX) и создание красивого визуала интерфейсов (UI).

Что делают?

  • Анализ пути пользователя
  • Создание кликабельных прототипов
  • Отрисовка макетов для разработчиков

Стек технологий

Figma Adobe XD Photoshop Illustrator Principle

Плюсы

  • Творческая работа
  • Сразу видишь визуал. результат

Минусы

  • Правки от заказчиков
  • Много общения с клиентами
Подробнее 100k - 350k ₽
